# 📚 Python Book Price Scraper

Projeto em Python que coleta preços de livros da web usando **Web Scraping** e exporta os dados para uma planilha Excel.

---

## 🚀 Tecnologias utilizadas

- Python
- Requests
- BeautifulSoup
- Pandas
- Excel (xlsx)

---

## 📊 O que o projeto faz

Este script:

1. Acessa um site de livros
2. Coleta os nomes e preços
3. Converte os preços para Real
4. Salva os dados em uma planilha Excel automaticamente
